__init__(self, fname, tgraph)
(Constructor)

source code 

Initialises the converter. If a .nt file is given, the converter uses this as an input to create a .csv file. However, if a directory is specified, then all .nt files in this directory are converted to .csv files.

Parameters:
  • fname - The input NTriples file (or directory with NTriples files).
  • tgraph - The target graph URI.
